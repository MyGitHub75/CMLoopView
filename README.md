# CMLoopView
简易的图片轮播图
## 屏幕截图

![](https://github.com/MyGitHub75/CMLoopView/blob/master/ScreenShot/loopView.gif?raw=true">)

# 系统支持

* iOS7+

* Xcode7

## 安装 
### CocoaPods

* 在 Podfile 中输入以下内容：
```
pod 'CMLoopView'
```
### 其他方法
* 下载文件包，将包（CMLoopView）拖入程序

## 使用
引入头文件做一下相关操作即可，详情见Demo
```
#import "CMLoopView.h"
```
